# MyWeatherDashboard

## Purpose
This dashboard runs in the browser and feature dynamically updated HTML and CSS and uses OpenWeather One Call API to retrieve weather data for cities.

## Built With
* HTML
* CSS
* Javascript
* OpenWeather One Call API
